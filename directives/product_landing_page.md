# Diretiva: Product Landing Page

## Objetivo
Criar uma landing page moderna, elegante e minimalista para portfólio profissional de QA Engineer.

## Especificações Visuais
- **Tema**: Dark Mode obrigatório.
- **Cores**:
  - Fundo Principal: `#0B0B0D` (Preto Profundo)
  - Fundo Secundário: `#111113` (Cinza muito escuro para seções alternadas)
  - Accent Principal: `#5A0F1C` (Vinho Escuro)
  - Accent Hover: `#7A1B2C`
  - Texto Principal: `#EAEAEA`
  - Texto Secundário: `#9CA3AF`
- **Tipografia**: Hierárquica, limpa, moderna (e.g., Inter, Roboto ou Outfit).
- **Estilo**: Corporativo, sério, premium. Sem cores vibrantes ou elementos infantis.

## Estrutura da Página
1.  **Navbar**: Fixa, minimalista. Links para seções.
2.  **Hero**: 
    - Manchete forte sobre QA/Automação.
    - Subtítulo breve.
    - Imagem profissional lateral direita (ou abstrata tecnológica).
    - CTA principal.
3.  **Sobre**: Breve bio profissional.
4.  **Skills**: Grid de habilidades (Linguagens, Ferramentas, Frameworks).
5.  **Projetos**: Cards com cases de sucesso (Problema, Solução, Resultado).
6.  **Vídeos**: Seção para demos de automação (placeholders ou embeds).
7.  **Metodologia**: Como trabalho (Processos, CI/CD, etc).
8.  **Prova Social**: Depoimentos ou logos de empresas.
9.  **CTA Final**: Chamada para contato/contratação.
10. **Footer**: Links sociais, copyright.

## Comportamento
- Responsivo (Mobile First).
- Animações suaves ao rolar (scroll reveal).
- Hover effects elegantes em botões e cards.

## Tecnologias
- HTML5 Semântico.
- CSS3 (Vanilla, uso de Variáveis CSS).
- JavaScript (Vanilla, sem frameworks pesados).
